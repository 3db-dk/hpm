# Architecture Sync Command

Synchronizes architectural documentation with current codebase implementation.

## Usage
```
/architecture-sync
```

## Description
Reviews current codebase structure and updates architectural documentation to reflect actual implementation. Ensures documentation accuracy and identifies architectural drift.

## Process
1. Analyze current module structure and dependencies
2. Review public API surfaces and interfaces
3. Compare with existing architectural documentation
4. Identify discrepancies and outdated information
5. Update documentation to reflect current state
6. Generate architectural decision records for changes

## Output
- Updated architecture documentation
- List of identified discrepancies
- Recommendations for architectural improvements
- Decision records for major changes
